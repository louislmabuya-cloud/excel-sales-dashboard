# Excel Sales Performance Dashboard

## Project Overview

This project is an interactive Excel sales dashboard created to analyse revenue, costs, profit, order volumes, products, and regional performance.

The project was originally completed as part of a guided Excel tutorial and was later independently reviewed, corrected, reorganised, and improved for portfolio use.

The dataset is fictional and contains 40 sales orders recorded from January to August.

## Project Objectives

The dashboard was created to:

* Track total revenue, total cost, total profit, and order volume.
* Compare monthly revenue and costs.
* Analyse sales performance by region.
* Compare revenue generated by different products.
* Analyse revenue from standard and bulk orders.
* Present sales information in a clear and visually accessible format.

## Data Improvements

During the review process, several inconsistencies from the original tutorial workbook were corrected:

* Corrected inconsistent product revenue values.
* Recalculated revenue using quantity and standard product prices.
* Replaced the fixed cost deduction with product-based cost calculations.
* Added regional shipping-cost calculations.
* Removed Grand Total values from charts.
* Corrected regional analysis to include all regions.
* Renamed unclear columns and workbook sections.
* Reorganised the workbook into a compact, portfolio-ready format.

## Key Insights

* Laptops generated the highest total revenue.
* Bulk orders generated more revenue than standard orders.
* The North region generated the highest regional revenue.
* Monthly revenue changed significantly across the eight-month period.
* Profit margins varied by product because each product used a different cost rate.
